#LostNet - Sistema Distribuido de Geolocalización y Reportes

> ** ADVERTENCIA LEGAL IMPORTANTE**: Lea la sección de Licencia al final de este documento antes de interactuar con este código.

##Descripción
LostNet es una plataforma integral diseñada para la gestión de reportes ciudadanos y geolocalización en tiempo real. Este proyecto implementa una arquitectura de **Sistemas Distribuidos** robusta, integrando servicios web, aplicaciones móviles y almacenamiento seguro.
